# Express API

## Description
This project is a simple **REST API** built using **Express.js**.

The API allows communication between the frontend and backend by sending and receiving data using HTTP requests such as:

- GET → retrieve data
- POST → create data
- PUT → update data
- DELETE → remove data

This project can be used as a **starter template** for building APIs with Node.js and Express.

---

## Features

- RESTful API structure
- CRUD operations (Create, Read, Update, Delete)
- JSON request and response handling
- Organized folder structure
- Easy to configure
- Beginner friendly

---

## Technologies Used

- Node.js
- Express.js
- JavaScript
- Database (PostgreSQL / MySQL / MongoDB / SQLite)
